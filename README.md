# ⚡ Digital Maintenance Management System - Assiut South Sector
**Developed by: Eng. Osama Sadek Ibrahim Saweers**
*Senior Infrastructure Electrical Engineer & Maintenance Manager*

## 🚀 Project Overview
This is a custom-built automation tool designed for the **Middle Egypt Electricity Distribution Company (MEEDC)**. It digitizes the reporting workflow for 42 different maintenance categories in the electrical grid.

## 🛠️ Tech Stack
- **Language:** Python 🐍
- **Framework:** Gradio (Web Interface)
- **Data Storage:** Excel (Automated Reporting)
- **Deployment:** Hugging Face Spaces

## 📈 Impact
- Real-time field reporting with GPS and Multi-photo documentation.
- Automated data aggregation for 12 engineering districts.
- 100% reduction in manual paperwork for maintenance logs.
